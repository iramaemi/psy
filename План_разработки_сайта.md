## 3. Дизайн-система

### 3.1. Шрифты

| Назначение | Шрифт | Начертание |
|------------|-------|------------|
| Заголовки (H1, H2) | Bonche Light | Light |
| Основной текст, навигация, подписи | AA Stetica | Regular |

> **Важно:** Шрифты находятся в Font Book пользователя. Для веба необходимо конвертировать в форматы `.woff2`, `.woff` и подключить через `@font-face`.

### 3.3. Модульная сетка

| Параметр | Значение |
|----------|----------|
| **Ширина модуля** | 66 px |
| **Высота модуля** | 72 px (4 строки) |
| **Высота строки** | 18 px |
| **Размер микромодуля** | 18 × 18 px |
| **Модули по ширине** | 12 шт. + 11 микромодулей |
| **Модули по высоте** | 36 шт. + 37 микромодулей |
| **Общая ширина body-wrapper** | 12×66 + 11×18 = 990 px |
| **Общая высота body-wrapper** | 36×72 + 37×18 = 3258 px |

**Расположение по ширине:** 1 модуль → 1 микромодуль → 2 модуль → … → 11 микромодуль → 12 модуль

**Расположение по высоте:** 1 микромодуль → 1 модуль → 2 микромодуль → … → 36 модуль → 37 микромодуль

### 3.4. Page-inner (контентная область)

| Параметр | Значение |
|----------|----------|
| **По ширине** | со 2 модуля до 11 модуля |
| **По высоте** | с 1 микромодуля до 11 микромодуля (относительно секции) |
| **Border-radius** | 1 ширина модуля = 66 px |

## 9. Примечания

1. **Пунктирные круги:** реализовать через SVG (`stroke-dasharray`) или CSS (`border-style: dashed` с `border-radius: 50%`). Рекомендуется SVG для точного контроля толщины и вида пунктира.
2. **Розовые овалы:** реализовать через `border-radius` с большим значением (пилюлевидная форма) или SVG.
3. **Зелёные прямоугольники (Услуги):** `border-radius: 66px` (или больше для полной скруглённости).
4. **Фото с прозрачностью:** убедиться, что PNG корректно отображаются на фоне `page-inner`.
5. **Сердечко в заголовке:** вставить `heart.png` как inline-изображение в тексте заголовка.
6. **Масштабирование:** ключевая особенность дизайна — все элементы должны масштабироваться пропорционально при изменении разрешения.
