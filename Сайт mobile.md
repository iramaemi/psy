# Сайт mobile  
**Промт**:  
**==Общая информация:==**  
Мобильная версия сайта - mobile.  
++Шрифты++: Bonche Light и AA Stetica Regular и Bold, которые лежат в Fonts Book.  
++Как должен выглядеть сайт++ со всеми шрифтами, цветами, масштабом и расположением элементов, можешь взять из /Users/irinavinokurova/Documents/Работа/PSY/_Сайт/Images_png с названием website mobile.png  
++Как выглядит сайт с сеткой:++  
/Users/irinavinokurova/Documents/Работа/PSY/_Сайт/Images_png с названием website mobile GRID.png  
Сетка десктопной и мобильной версии имеет тот же масштаб, только мобильная версия имеет «укороченное» поле сетки.  
Все элементы мобильной версии повторяют по шрифтам, цветам, настройкам элементов десктопную версию. Отличается только расположение в сетке.  
++Все картинки png++: /Users/irinavinokurova/Documents/Работа/PSY/_Сайт/Images_png  
  
Все элементы страницы будут располагаться относительно сетки.  
Масштабируй все элементы относительно друг друга, придерживаясь сетки и расположения элементов. Если будет больше или меньше разрешение монитора, то увеличивай или уменьшай все элементы относительно друг друга, придерживаясь масштаба и сетки, что я дам.  
++Данные сетки:++  
Layout width 318 пикселей.  
Модуль width 66 пикселей х высота 72.  
Module height 4 строчки.  
Высота строчки 18.  
Размеры микромодуля 18 х 18.  
++Сетка по ширине располагается так:++  
1 модуль - 1 микромодуль - 2 модуль …… - 3 микромодуль - 4 модуль  
++Сетка по высоте располагается так:++  
1 микромодуль - 1 модуль - 2 микромодуль ……. - 34 модуль - 35 микромодуль  
++Размер сетки:++  
Модули: по ширине 4 штук х по высоте 34 штук.  
Микромодули: по ширине 3 х по высоте 35 штук.  
++Расположение общих элементов страницы:++  
• body-wrapper:  
По размерам сетки  
• page-inner:  
Примыкает к верхнему краю страницы.  
По ширине: 3 модуля + 3 микромодуля  
По высоте: 31,5 модуль + 32 микромодуля  
Закругление углов: диаметр 1,5 строчки  
  
Цвета: все цвета элементов такие же, как цвета в десктопной версии.  
Толщина линий: такие же, как в десктопной версии.  
  
Далее все элементы в секциях будут расписаны по расположению относительно данной сетки, начиная с левого верхнего микромодуля (микромодуль 1).  
Все секции будут находится внутри границ page-inner, кроме секции контакты (как в десктопной версии контакты будут на сером фоне body-wrapper).  
  
Обрати внимание, что некоторые изображения нужно заменить на соответствующую версию (mobile).  
  
**==Всего будет 7 «секций»:==**  
Все элементы в секциях перечислены сверху вниз и слева-направо, если не указано иное расположение.
